# REST API Test Automation Framework

Automated API test suite validating CRUD operations, authentication flows,
and negative/edge case scenarios against a live REST API using Python and pytest.

## Tech Stack
- Python 3.x
- pytest
- requests
- pytest-html

## Features
- CRUD API Testing
- Authentication Testing
- Negative Testing
- Data-driven Testing
- HTML Report Generation

## Run Tests

```bash
pip install -r requirements.txt
pytest
